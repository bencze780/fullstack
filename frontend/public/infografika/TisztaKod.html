Tiszta Kód Alapelvek

"A jó kód olyan, mint egy jól megírt könyv: olvasás közben azonnal érted a történetet."

Beszédes nevek

A változó neve legyen leíró. Aki olvassa, tudja meg belőle, mit tárol és mi a célja.

Egy feladat (SRP)

Egy függvény csak egyetlen dolgot csináljon. Ha "és" kötőszó kell a leírásához, szedd szét!

DRY Elv

"Don't Repeat Yourself". Ne legyen kódduplikáció. Használj közös segédfüggvényeket.

Kevesebb komment

A kód legyen önmagát dokumentáló. Csak a "miért"-et kommenteld, ne a "hogyan"-t.

A kommentelés evolúciója

Nézzük meg, hogyan teszi feleslegessé a jó elnevezés a szájbarágós kommenteket ugyanannál a logikánál.

function calc(a, b) {
  // Kiszámoljuk a szorzatot
  let res = a * b;
  // Visszaadjuk a kiszámolt értéket
  return res;
}

Kifejező Logika 
function sumProductPrices(products) {
  return products.reduce((total, product) => {
    return total + product.price;
  }, 0);
}
