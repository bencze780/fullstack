🧪 Webes Teszt Futtató

Kattints egy gombra a backend tesztek futtatásához. A folyamat eltarthat néhány másodpercig.

💡 Tesztek módosítása

A tesztek módosításához a backend mappában található tesztfájlokat kell szerkesztened. A kód módosítása és elmentése után elegendő csak újra megnyomni a fenti futtatás gombok valamelyikét.

⚠️ Fontos: A tesztek webes felületről történő indításához a backend szervernek mindenképpen futnia kell!

📝 Példa: A backend/sum.test.js kiegészítése egy új teszttel

const sum = require('./sum');

test('összeadja az 1 + 2-t, hogy 3-at kapjunk', () => {
    expect(sum(1, 2)).toBe(3);
});

// Próbáld meg hozzáadni ezt az új tesztet a fájl aljára:
test('összeadja a 10 + 5-öt, hogy 15-öt kapjunk', () => {
    expect(sum(10, 5)).toBe(15);
});
Kattints egy fenti gombra a tesztelés megkezdéséhez...
