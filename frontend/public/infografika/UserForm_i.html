UserForm.jsx

"A prezentációs beviteli felület"

"Buta" komponens koncepció

A UserForm nem tud az adatbázisról. Csak összegyűjti az adatokat (local state) és értesíti a szülő komponenst, ha készen van az űrlap elküldésére.

Belső Állapotok

Name (useState)

Minden leütésre frissül az onChange segítségével.

Email (useState)

React által kontrollált input mező.

Külvilágtól kapott Propok

onAddUser isSubmitting

Adatáramlás útja

1

Felhasználó kitölti az inputokat.

2

Submit (Prevent Default lefut).

3

Callback hívása az adatokkal.

4

Mezők kiürítése (reset state).
