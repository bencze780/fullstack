App.jsx

"Az alkalmazás központi agya és vezérlőegysége"

Állapotkezelés

Felhasználók listája, betöltési állapotok, hibák és szerkesztési ID tárolása.

Adatkezelés

HTTP kérések (CRUD) indítása az API felé (axios), adatok szinkronizálása.

Koordináció

Prop-ok és callback függvények átadása a gyerek komponenseknek.

Kezelt Állapotok (useState)

users Array

A backendről kapott felhasználók listája.

loading Boolean

Betöltés jelzése a lekérés alatt.

isSubmitting Boolean

Gombok letiltása mentés közben.

error String | null

Hibaüzenetek tárolása és kijelzése.

editingId Number | null

Az éppen szerkesztett felhasználó azonosítója.

API Műveletek

  • fetchData: GET /api/users
  • handleAddUser: POST /api/users
  • handleUpdate: PATCH /api/users/:id
  • handleDelete: DELETE /api/users/:id

UI Logika

  • handleEditStart: Szerkesztési mód BE
  • handleEditCancel: Szerkesztési mód KI
  • Logikai sorrend: Loading → Error → Content
